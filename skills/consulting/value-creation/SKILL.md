# Value Creation Consulting Frameworks Hub

11 frameworks from A&M, PwC, and Bain for post-acquisition value creation planning, cost optimization, and initiative tracking.

> Before executing, read `../../_shared/plugin-context.md` for Nexus plugin conventions.

## Prerequisites

- **Nexus MCP** (optional): `nexus_list_portfolio`, `nexus_get_deal` for portfolio company data
- **Manual fallback**: Provide company financials, cost structures, and initiative plans directly

## Common Value Creation Workflow

```
100-Day Plan (structure)
  --> Value Creation Planner (quantify)
  --> Cost Optimization + Working Capital + Pricing (execute)
  --> PEPI Tracker (monitor)
  --> Results Delivery (report)
```

## 100-Day Plan Template

| Phase | Days | Focus | Deliverables |
|-------|------|-------|-------------|
| Stabilize | 1-30 | Quick wins, team assessment, data gathering | Baseline assessment, quick-win list, org chart review |
| Accelerate | 31-60 | Initiative launch, process redesign, governance | Initiative charters, KPI dashboards, steering committee |
| Sustain | 61-100 | Full execution, tracking, course correction | EBITDA bridge update, management scorecard, board report |

## EBITDA Bridge Template

```
Current EBITDA: $XXM
  + Revenue enhancement
    - Pricing improvements:     $X.XM
    - Cross-sell / upsell:      $X.XM
    - New market entry:         $X.XM
  + Cost reduction
    - Procurement savings:      $X.XM
    - Headcount optimization:   $X.XM
    - Facility rationalization: $X.XM
  + Working capital release
    - DSO reduction:            $X.XM (one-time cash)
    - DPO extension:            $X.XM (one-time cash)
  - Implementation costs:       ($X.XM)
Target EBITDA: $XXM
```

## Working Capital Metrics

| Metric | Formula | Benchmark |
|--------|---------|-----------|
| DSO | (AR / Revenue) x 365 | Industry median |
| DPO | (AP / COGS) x 365 | Industry median |
| DIO | (Inventory / COGS) x 365 | Industry median |
| CCC | DSO + DIO - DPO | Lower = better |
