# Corporate Transformation Frameworks Hub

5 frameworks for corporate turnaround, transformation, and organizational change.

> Before executing, read `../../_shared/plugin-context.md` for Nexus plugin conventions.

## Prerequisites

- **Nexus MCP** (optional): `nexus_list_portfolio`, `nexus_get_deal` for portfolio company context
- **Manual fallback**: Provide company financials, org charts, operating data, and transformation objectives directly

## Framework Registry

| Framework | Origin | Purpose |
|---|---|---|
| **Turnaround** | BCG TURN + A&M | Three-phase turnaround with 13-week cash flow model |
| **Corporate Transformation** | A&M + Deloitte | Assess current vs target operating model across all dimensions |
| **Operating Model Design** | McKinsey | Design target operating model across 5 dimensions: process, org, tech, governance, people |
| **Finance Transformation** | Deloitte | Assess close cycle, FPA reporting, and exit-readiness of finance function |
| **Workforce Transformation** | PwC | Assess org readiness, map talent risk, create change management plans |

## Common Transformation Workflow

```
Turnaround (stabilize)
  --> Corporate Transformation (redesign)
  --> Operating Model Design (target state)
  --> Finance/Workforce Transformation (execute by function)
```

## Turnaround -- BCG TURN Framework

| Phase | Duration | Focus | Key Deliverables |
|-------|----------|-------|-----------------|
| **Triage** | Weeks 1-2 | Cash position, liquidity, immediate risks | 13-week cash flow model, stakeholder map |
| **Stabilize** | Weeks 3-8 | Cash management, cost cuts, revenue protection | Cash preservation plan, headcount actions |
| **Transform** | Weeks 9-26 | Structural changes, new operating model | Restructuring plan, value creation roadmap |
